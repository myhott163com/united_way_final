# UWSM Community Data Analysis

Analysis of the United Way of Southern Maine community survey (1,889 respondents). Covers data cleaning, NLP classification of open-ended responses, descriptive statistics, and statistical modeling.

## Contents

- `01_cleaning_processing.ipynb` loads the raw survey, cleans responses, classifies open-ended text, and writes a processed CSV.
- `02_analysis.ipynb` runs descriptive stats, chi-square tests, logistic regression, a decision tree, and PCA.
- `requirements.txt` lists Python dependencies.

## Setup

Python 3.9 or newer is required.

1. Create a virtual environment and activate it:
   ```bash
   python -m venv venv
   source venv/bin/activate        # Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Download the spaCy language model (pip alone will not install this):
   ```bash
   python -m spacy download en_core_web_sm
   ```

## Data files

Data files are not tracked in this repo. Place them in the project root, next to the notebooks:

- `UWSM Community Data 4-1-2026.xlsx` (raw survey, required for the cleaning notebook)
- `processed_data_5.csv` (cleaned dataset generated by `01_cleaning_processing.ipynb`, used as input by `02_analysis.ipynb`)

If you only have the raw Excel file, run the cleaning notebook first to generate the CSV.

## Running the notebooks

Run in order:

1. `01_cleaning_processing.ipynb`. Expect 10 to 30 minutes on a GPU and several hours on CPU. The BART zero-shot classifier downloads about 1.6 GB on first run.
2. `02_analysis.ipynb`. Runs in a few minutes.

If `processed_data_5.csv` is already in place, you can skip straight to the analysis notebook.
